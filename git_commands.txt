** Git commands **

git status
git add/rm <file>
git commit -m "Commit message."

git push
or
git push -u origin <branch_name>

git branch <branch_name2>        # create a new branch           Ex.: git branch dev
git checkout <branch_name2>      # switch to the <branch_name2>  Ex.: git checkout dev

When you want to merge your feature branch (dev) into the master branch:

git checkout master
git fetch
git merge dev
